# MySTL
STL-like C++ library.

## Data Structures
My-STL provides out-of-the-box data structures, and is a combination of existing C++ STL data structures, data structures with improved performance and data structures that haven't been implemented in the C++ STL.
### Standard
- Disjoint Set
- Heap
- List
- Queue
- Stack
- Vector
### Graphs
- Abstract Graph
- Adjacency List
- Adjacency Matrix
- Directed Graph
- Graph Adjacency Base
- Undirected Graph
### Maps
- Chained Map
- Dictionary
- Double HashMap
- Open Map
### Trees
- Binary Search Tree
- AVL Tree
- Red-Black Tree
